Even though this is a pretty pedestrian visualization, it's pretty cool to give those enrollment numbers geometric meaning. That's a lot of biology majors... How It's Done The program bundles together the department code with the number of undergraduates enrolled in it, then generates a circle with a random color with size proportional to the number of students.  The scale factor that controls the circle size is  constant within each rendering, but varies between renderings.
